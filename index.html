原项目 node-unblocker

node-unblocker是一个由nfredly开发的Node.js网页代理项目

但是原项目没有强制跳转https功能，这使其安全性大大减小

所以这是一个教你如何在VPS上部署有跳转https功能的node-unblocker的教程

第一步

git clone https://github.com/zhangke200377/heroku-node-proxy.git 克隆此仓库

第二步

安装相关程序 curl -sL https://deb.nodesource.com/setup_10.x -o nodesource_setup.sh

sudo bash nodesource_setup.sh

sudo apt install nodejs -y

sudo apt install build-essential -y

第三步

cd heroku-node-proxy

npm install

npm start

第四步

用Nginx反向代理127.0.01:8080
